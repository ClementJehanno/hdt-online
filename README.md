# HDT online

A simple [HDT](http://www.rdfhdt.org/ "rdfhdt") converter online service.

## Usage

	$ python hdt-online.py -b abc.com -p 8080

## Dependencies

* [HDT](http://www.rdfhdt.org/), download and build [HDT Java](http://code.google.com/p/hdt-java/)
